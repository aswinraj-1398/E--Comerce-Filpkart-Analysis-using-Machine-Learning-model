# E-Commerce Flipkart Analysis Using Machine Learning Model

## Overview
This project focuses on analyzing e-commerce data from Flipkart using machine learning techniques. The analysis includes data scraping, exploratory data analysis (EDA), feature engineering, and building predictive models to extract valuable insights from e-commerce product data.

## Project Objectives
- Scrape and collect e-commerce product data from Flipkart
- Perform comprehensive exploratory data analysis (EDA)
- Build and evaluate machine learning models for predictions
- Generate actionable insights for e-commerce business decisions

## Dataset
The project uses e-commerce product data from Flipkart, including:
- Product names and categories
- Prices and discounts
- Ratings and reviews
- Product specifications
- Availability status

## Technologies & Libraries Used
- **Python 3.x**
- **Pandas** - Data manipulation and analysis

- **Matplotlib & Seaborn** - Data visualization

## Installation & Setup

### Prerequisites
- Python 3.7 or higher
- pip (Python package manager)

### Steps
1. Clone the repository:
```bash
git clone https://github.com/aswinraj-1398/E--Comerce-Filpkart-Analysis-using-Machine-Learning-model.git
cd E--Comerce-Filpkart-Analysis-using-Machine-Learning-model
```

2. Create a virtual environment (optional but recommended):
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. Install required dependencies:
```bash
pip install -r requirements.txt
```

4. Open the Jupyter Notebook:
```bash
jupyter notebook notebooks/E-commerce-Flipkart-Analysis.ipynb
```
